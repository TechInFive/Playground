# PyGame Playground 🐍

## Features 🎮

- Clock: A traditional clock

## Prerequisites 🛠

- Python 3.x
- PyGame library

## Setup & Run 🚀

1. Clone the repository:
```
git clone https://github.com/TechInFive/Playground
cd Playground
```

2. Install the required packages:
```
pip install pygame
```
